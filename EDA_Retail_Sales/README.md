# EDA on Retail Sales Data

## Objective
To perform exploratory data analysis on retail sales data and extract meaningful insights to support business decisions.

## Dataset
Retail Sales Dataset  
https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Data cleaning and preprocessing
- Descriptive statistics
- Time series analysis
- Customer and product analysis
- Data visualization

## Insights
- Identified sales trends over time
- Analyzed customer purchasing behavior
- Found top-performing product categories

## Conclusion
EDA helps businesses understand sales patterns and improve decision-making.
